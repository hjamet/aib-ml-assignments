⚠️  EN CAS DE PROBLÈME
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Application lente :
   1. Demander aux élèves : F5 (rafraîchir)
   2. Faire une pause (réduire la charge)

Application bloquée :
   1. Ctrl+C dans le terminal
   2. Attendre 5 secondes
   3. ./lancer_cours.sh (relancer)
   4. Élèves : F5 dans le navigateur

Connexion impossible :
   1. Vérifier firewall : sudo ufw allow 8501
   2. Vérifier même réseau WiFi
   3. Vérifier IP (192.168.x.x pas 127.0.0.1)
